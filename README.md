# Epoch

Clock & Calendar - Multi-timezone world clock and calendar menu bar app for macOS.

## Features

- Multiple world clocks with timezone support
- Drag and drop to reorder clocks
- Click clock to copy time to clipboard
- Integrated calendar with month navigation
- Click calendar to open system Calendar.app
- Time difference indicator (+5h, -3h, etc.)
- Keyboard shortcut (Cmd+Shift+C) to toggle window
- Font size options (Small, Medium, Large)
- Layout options (Horizontal, Vertical)
- Menu bar icon customization (Time, Date, Month+Day)
- Persistent clock storage with localStorage
- Duplicate timezone validation
- Dark and Light themes
